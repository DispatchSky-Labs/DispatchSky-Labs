Frequently Asked Questions (FAQs)

What is SkyNotify?

SkyNotify provides real-time aviation weather updates (METARs, TAFs, SIGMETs) and customizable alerts for aircraft dispatchers and pilots. It’s for informational use only—verify with FAA sources.

Where does the weather data come from?

Data is sourced from NOAA’s Aviation Weather Center and licensed APIs, ensuring reliable updates.

How often is weather updated?

METARs and TAFs refresh every 30–60 minutes, based on source issuance. Alerts are pushed in real-time.

Why isn’t data loading?

Check internet, location permissions, or refresh the app. Email support@skynotify.com if issues persist.

Is SkyNotify FAA-approved?

No, SkyNotify is an informational tool, not FAA-certified. Use sources like 1800WXBRIEF for operational decisions.

How do I set up alerts?

Go to Settings > Alerts, select airports/routes, set conditions (e.g., wind, visibility), and enable push notifications.
